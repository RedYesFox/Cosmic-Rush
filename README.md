# Cosmic Rush

Справка по программе

## Описание проекта

* **Cosmic Rush** — это казуальная игра в жанре аркады.
* В этой игре вы управляете **космическим кораблем**, который должен защищаться от **враждебных инопланетян**.
  У вас есть орудие, с помощью которого вы можете стрелять по инопланетянам;
* Ваша задача — продержаться как можно дольше и набрать наибольшее количество очков;
* Категории игроков, которым может понравиться такая игра:
    1) Казуальные игроки;
    2) Любители космической тематики;
    3) Поклонники аркадных игр.

## Техническое задание

#### Цели и задачи

- Создать стартовое меню и меню паузы;
- Создать игровое окно (игрок внизу экрана, враги появляются в верхней части экрана);
- Реализовать функционал для передвижения игрока, его действий;
- Обеспечить интерфейс информацией о здоровье игрока, его броне, текущем уровне и количестве набранных очков;
- Создать вражеские корабли и настроить их поведение;
- Реализовать корректировку сложности игры в зависимости от уровня, до которого дошел игрок;
- Реализовать функционал для сохранения лучшего результата игрока;
- Интерфейс должен быть интуитивно понятным и удобным для пользователя.

#### Особенности

- Стартовое меню содержит информацию о рекорде, достигнутом игроком
- Всплывающее окно с подсказками об управлении
- Система очков за уничтожение врагов
- Здоровье героя, уменьшающееся при столкновении со снарядами врага
- Увеличение сложности с каждым уровнем (увеличение количества врагов)
- Отображение текущего счета, уровня и здоровья на экране
- Экран Game Over

#### Как играть

1. Запустите игру
2. Нажав на кнопку "Начать игру", вы отправляетесь в бой
3. Используйте Мышь, двигая ее влево и вправо, перемещайте героя
4. Нажимайте ЛКМ для стрельбы из орудия
5. По готовности брони, можно ее активировать нажав на Пробел
6. Уничтожайте вражеских захватчиков и набирайте очки
7. Следите за здоровьем корабля и избегайте снарядов
8. Игра заканчивается, когда здоровье героя достигает нуля
9. Во время игры можно выйти, но счет не сохранится

## Требования к установке

- **Язык программирования:** Python 3.12
- **Библиотеки:** Pygame, plyer,
- **OC:** проект создавался на базе ОС Windows 10

## Всплывающее окно

После нажатия кнопки старта игры сплывает окно с подсказками об управлении
<p align="center">
  <img src="/For Readme/TipsWindow.png" alt="picture" style="width:600px;"/>
</p>

В **Cosmic Rush** игрок может совершать всего несколько действий:

1. Нажатие ЛКМ - Огонь по врагам;
2. Нажатие Пробела - Активация брони;
3. Клавиша ESC отвечает за паузу и выход из неё (а также за открытие меню выхода из программы);
4. Клавиши E и R активны не всегда: E только во время паузы и отвечает за закрытие программы, а R перезагружает игру,
   после смерти игрового персонажа.

# Перспективы развития

После выполнения основного технического задания, в программу планируется добавить следующий **функционал**:
- Переключение полноэкранного-оконного режима
- Назначение своих биндов
- Усложнение игры (Добавление боссов, случайных событий)
- Выпадение бонусов из врагов
